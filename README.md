# LIA Tracker

A lightweight, client-side internship application tracker. No backend, no account needed — all data is stored in your browser's `localStorage`.

## Features

- Add, edit, and delete internship applications
- Track status: Applied, Interview Scheduled, Interviewed, Offer, Rejected, Withdrawn
- Stats dashboard (total, applied, interviews, offers, rejected)
- Search by company or role
- Filter by status
- Sort by date, company name, or status
- Color-coded status badges and left-border accents
- Dark purple UI, fully responsive

## Usage

Just open `index.html` in any modern browser — no build step required.

```
git clone https://github.com/YOUR_USERNAME/LIA_Tracker.git
cd LIA_Tracker
# Open index.html in your browser
```

## Tech Stack

- Plain HTML / CSS / JavaScript
- `localStorage` for persistence
- Zero dependencies

## Deployment

You can host it for free on **GitHub Pages**:

1. Push to GitHub
2. Go to Settings → Pages
3. Set source to `main` branch, `/ (root)`
4. Your tracker will be live at `https://YOUR_USERNAME.github.io/LIA_Tracker`
